# www-dezhou-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，仅作为一组静态页面的集合，方便浏览和索引。

## 项目简介

本仓库维护了一批独立编写的 HTML 页面，内容涵盖信息整理、工具页面、参考手册等。每个页面均为自包含的 HTML 文件，无需额外依赖即可在浏览器中直接打开。仓库的主要目的是长期保存这些页面，并提供统一的访问入口。

## 目录说明

```
www-dezhou-guide/
├── README.md           # 本文件
├── index.html          # 仓库首页 / 导航页
├── pages/              # HTML 页面归档目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- **index.html**：仓库的入口页面，列出所有归档页面的链接。
- **pages/**：存放所有独立的 HTML 页面，每个文件均可单独访问。
- **assets/**：存放页面共用的静态资源，如图片、CSS 样式表和 JavaScript 脚本。

## 页面归档说明

- 所有页面均为纯前端实现，不依赖后端服务。
- 页面之间相互独立，没有跨页面耦合。
- 每个页面在 `pages/` 目录下以 `.html` 文件形式存在。
- 页面内容可能不定期更新或补充，历史版本可通过 Git 提交记录查看。

## 维护说明

- 欢迎通过 Issue 提交页面内容建议或问题反馈。
- 如需新增页面，请在 `pages/` 目录下创建新文件，并在 `index.html` 中添加链接。
- 请保持页面风格简洁，避免引入过多外部资源。
- 本仓库不接受任何形式的商业广告或推广内容。
